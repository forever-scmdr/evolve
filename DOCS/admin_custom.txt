Кастомные (отличающиеся от стандартных)

1.  Корневые страницы для групп пользователей кроме common описываются в файле pages.xml с помощью элементов
	admin-page с атрибутом group, равным названию соответствующей группы.
2.  Кроме корневых страниц групп пользователей с помощью элемента admin-page можно определять страницы для любых
	других администраторских страниц (страниц редактирования определенных айтемов). В этом случае нужно указать в
	атрибуте item элемента admin-page название нужного айтема, например, product.
3.  Поскольку для страницы редактирования айтема нужны 2 источника информации - сам айтем и его сабайтемы, то
	вновь определяемая страница должна содержать страничные айтемы с id равными item (базовый айтем) и
	subitems (список айтемов слева, т.е. в стандартном варианте - сабайтемы базового айтема).
	Список доступных для создания айтемов - это все доступные сабайтемы базового по всем ассоциациям.
