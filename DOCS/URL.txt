			ДОМЕН                               PATH "TRANSLIT"                    PATH переменные          QUERY



ДОМЕН               просто доменное имя

PATH "TRANSLIT"     Часть урла, в которой переменные PATH не имеют названия и являются уникальными текстовыми ключами
					соответствующих айтемов. Они всегда должны идти в самом начале урла. Соответствующие им переменные
					страницы должны идти в начале страницы (перед всеми другими) и в том же порядке, что и в урле.
					Ключем, определяющим страницу, является ПОСЛЕДНИЙ элемент этой части, в данном случае teplovizor_testo_865.
					Это справедливо даже в случае, если количество элементов в урле меньше количества key
					переменных страницы

PATH переменные     Часть PATH, в которой переменные имеют названия (brand - название, testo - значение)
					Эта часть всегда идет после части PATH "TRANSLIT"

QUERY               query часть урла
